# We the people

### Run Locally

1. Clone the project

    ```bash
    git clone https://github.com/MarkVed17/we-the-people.git
    ```

2. Go to the project directory

    ```bash
    cd we-the-people
    ```

3. Install dependencies

    ```bash
    npm install
    ```

4. Start the server

    ```bash
    npm run start
    ```
